## 绘制国庆头像

<center><img src="./output/ch_avatar.png"></center>

`python make_ch_avatar.py -i your_origin_avatar -o your_new_avatar -r 1`

-i: 输入头像路径

-o: 输出头像文件名，在output文件夹下

-r: 调整国旗大小，0.5～1.2

-s: 开启窗口，debug使用

**依赖**
opencv

numpy

`pip install -r requirements.txt`

`conda install --yes --file requirements.txt`
